ngx-codejar

An Angular wrapper for the code-editor CodeJar. It supports two-way data-binding.
For more information, issue requests or just saying thanks by giving a star see the Github repository

Below you see two examples of how to use ngx-codejar - one for highlight.js and one for prism.js. Each code block is an editor, you can change it!
